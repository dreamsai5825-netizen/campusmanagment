# **App Name**: CampusConnect

## Core Features:

- User Authentication: Secure login system with separate roles for Admin, Teachers, and Students.
- Admin Class Management: Admin panel for assigning teachers to classes.
- Attendance Monitoring: Admin can monitor student attendance across different classes.
- Teacher Dashboard: Teacher dashboard with cards displaying total students, assigned classes, timetable, and notifications.
- Teacher Profile: Teachers can update their profiles.
- Coursework Management: Teachers can assign homework and assignments to specific classes and students.
- Attendance Tracking: Class-wise attendance system where teachers can mark student attendance.
- Parent Communication: Enable parents to contact teachers.
- Announcements: Teachers and admins can send announcements/notifications to students and other teachers.

## Style Guidelines:

- Primary color: Deep sky blue (#33A8FF) to suggest collaboration and communication.
- Background color: Light blue (#E5F3FF) for a calm, clean interface.
- Accent color: Royal blue (#4169E1) to highlight key actions and elements.
- Body and headline font: 'PT Sans', a modern yet warm font, is used both for headlines and body text for clear readability.
- Use clear and consistent icons for navigation and actions.
- Left sidebar for navigation and card-based layouts for dashboards and class displays.
- Subtle transitions and animations for user interactions (e.g., opening menus, displaying notifications).
